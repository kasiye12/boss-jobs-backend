# Boss Jobs Ethiopia - Backend API

Professional job portal backend tailored for the Ethiopian market, built with Node.js and PostgreSQL.

## Features

- 🚀 **High Performance**: Built with Express.js and PostgreSQL for scalability
- 🔐 **Secure Authentication**: JWT-based auth with Telegram Mini-App integration
- 📍 **Location-Based Search**: Geospatial queries for jobs near you
- 🤖 **Telegram Bot**: Integrated bot for job notifications
- 📱 **Offline-First**: JSONB structures for offline data sync
- 🎙️ **Voice Pitches**: Audio profile support for job seekers
- 📊 **Smart Matching**: AI-powered job recommendations
- 🐳 **Docker Ready**: Containerized for easy deployment

## Tech Stack

- **Runtime**: Node.js 18+
- **Framework**: Express.js 4.19
- **Database**: PostgreSQL 15 with PostGIS
- **ORM**: Sequelize 6
- **Cache**: Redis
- **Auth**: JWT + Telegram OAuth
- **Storage**: AWS S3 / DigitalOcean Spaces
- **Deployment**: Docker + PM2
